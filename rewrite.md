# /rewrite

Viết lại copywriting cho 1 hoặc nhiều sections trong LP.

## Usage

```
/rewrite [tên-file] [section: hero|problem|solution|benefits|faq|cta|all]
```

## Khi nào dùng
- Copy hiện tại chưa đủ mạnh
- Muốn thử tone/angle khác
- User cung cấp thêm info mới về sản phẩm
- Chuyển từ tiếng Việt → tiếng Anh hoặc ngược lại

## Execution

1. Đọc file LP + xác định section cần rewrite
2. Phân tích copy hiện tại: điểm mạnh, điểm yếu
3. Viết lại theo framework:
   - **Hero**: Pain-first HOẶC Outcome-first (chọn cái phù hợp hơn)
   - **Problem**: Customer voice, ngôi thứ 2, cụ thể
   - **Solution**: Feature → Benefit conversion
   - **CTA**: Action verb + outcome + urgency
   - **FAQ**: Objection → Reassurance pattern
4. Cho user chọn: version cũ hay mới
5. Apply vào file

## Rules

- Giữ nguyên structure HTML/CSS — chỉ thay text content
- Đưa ra 2 versions nếu user chưa rõ tone muốn dùng
- Không tự ý thêm/bớt sections
